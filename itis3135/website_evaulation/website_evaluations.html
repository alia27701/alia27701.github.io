Website Evaluations

In this page, we are going to talk about a well made website and why it is well made, and a badly made website and why it is badly made.

We will be using the crap design principals to describe both websites.

The Kombat Pavilion

The Kombat Pavilion was a fan website decidicated for the game series known as Mortal Kombat. It contains fanmade games, comics, and information about the lore of Mortal Kombat.

  • Contrast: This website has good contrast as the colors of the text is much brighter than the color of the background so it is easier to read.
  • Repetition: This website has good repetition as the colors, fonts, and font sizes are the same for each page of the website, making the website feel cohesive.
  • Alignment: This website has good alignment as the contents of the website can be read straight up to down, making it easy to read what is on the website.
  • Proximity: This website has good Proximity as the contents of the website is close together with enough space to tell what is different.

Atari Best Electronics

This is a website dedicated to selling replacement parts and accesories of Atari Systems. According to the website, they have 5000+ different Atari items, ranging from the 2600 to the Jaguar.

  • Contrast: The text size of this website is too similar to each other so it is hard to tell what is the title or body text of the website.
  • Repetition: The text color is not consistent as it randomly changes from black to blue. Throughout the website, different fonts are used as well, making the website not cohesive at all.
  • Alignment: The website is very sloppy as the text are too close to each other so most of the time, it is very hard to read and you are reading a wall of text.
  • Proximity: The text for the website is way to cluttered so it is hard to tell if the sections are related or not.
